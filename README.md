# UAV Attitude Control System (MATLAB / Simulink)

This project demonstrates the design and simulation of a PID based UAV attitude control system using MATLAB and Simulink.

The objective of the project was to design a control system capable of stabilising UAV roll and pitch dynamics while achieving acceptable transient response and system stability.

## Project Overview

The UAV dynamics were modelled and PID control loops were implemented to regulate aircraft attitude. Controller parameters were tuned and validated using simulation based step response analysis.

## Tools Used

MATLAB  
Simulink  
Control Systems Toolbox

## Key Features

PID controller design and tuning  
Attitude stabilisation simulation  
Step response and stability analysis  
Control system performance validation
